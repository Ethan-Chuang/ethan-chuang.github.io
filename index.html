Projects

California Home Price Prediction

Predicts single-family home sale prices from roughly a year of California MLS sold-listing data. Heavy feature engineering — price per bedroom, lot utilization, property age, ZIP-level averages, an amenity score — and a time-based split that trains on earlier months and tests on the most recent one, so the evaluation mirrors real forecasting.

Gradient Boosting reached R² 0.88 with a median error under 8% — half of predicted prices land within 8% of the true sale price.

Python scikit-learn XGBoost LightGBM CatBoost

Medicare Utilization Analysis

Inferential machine learning on public CMS Medicare Current Beneficiary Survey data. Condenses ~577 raw survey columns into a single continuous utilization score via PCA, predicts it with nine cross-validated regression models, then uses SHAP to explain which factors move the score and by how much.

The top three predictors were enrollment type, specialist access, and age — suggesting Medicare care strategies should be tailored by plan type rather than applied uniformly.

Python PCA XGBoost SHAP statsmodels
